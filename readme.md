# 单词记忆工具

一个基于Python和PyQt6开发的个人单词记忆工具，支持单词翻译、智能复习和云端同步。

## 功能特点

- 🌐 免费翻译功能（Google翻译、百度翻译等）
- 📚 智能间隔重复算法
- ☁️ Supabase云端存储
- 👥 多用户支持
- 🎨 简洁的苹果风格UI

## 快速开始

1. 安装依赖：`pip install -r requirements.txt`
2. 配置数据库：参见 docs/setup_guide.md
3. 运行程序：`python main.py`
